# Video Speed Keys

A Chrome extension for quickly changing the speed of YouTube videos with keyboard shortcuts. An overlay showing the video speed appears upon pressing any of the set keyboard shortcuts.

## Keyboard Shortcuts

| Key        | Description  |
| ---------- | ------------ |
| `-`        | Speed Down   |
| `+` OR `=` | Speed Up     |
| `*`        | Reset Speed  |
| `z`        | Show Overlay |
